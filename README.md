# Colibri_Chat

Repository skeleton created for the Colibri Chat project.

Structure:

- `apps/mobile/` — mobile application code
- `docs/` — project documentation
- `supabase/migrations/` — database migrations
- `supabase/policies/` — Row Level Security policies
- `supabase/functions/` — edge / serverless functions for Supabase

This repo currently contains placeholder `.gitkeep` files to ensure empty directories are tracked.
